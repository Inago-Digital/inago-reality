# Inago Reality

A marketing website for **Inago Reality** - a performance marketing service for real estate agents in the Czech Republic.

## Overview

This site promotes targeted social media advertising packages (Meta Ads) designed to help real estate agents sell properties faster by reaching potential buyers through Facebook and Instagram campaigns.

## Tech Stack

- [Eleventy](https://www.11ty.dev/) — Static site generator
- [React](https://react.dev/) — UI components (SSR + client hydration)
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [Framer Motion](https://www.framer.com/motion/) — Animations
- [TypeScript](https://www.typescriptlang.org/)

Template generated with [create-react-static](https://npmjs.com/package/create-react-static).
